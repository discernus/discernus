    Phase A: Data Extraction & Parsing

    Input: Database execution results from orchestrator Output: Structured analysis dataset

    Components: 1. Query Database Results: Extract all analysis results from experiment execution 2. Parse LLM Responses: Extract Dignity/Tribalism well scores from JSON responses 3. Data Validation: Ensure score completeness and format consistency 4. Multi-LLM Dataset: Structure for interrater reliability analysis

    Deliverable: experiment_analysis_dataset.csv with structured well scores

    Phase B: Statistical Analysis & Hypothesis Testing

    Input: Structured dataset with LLM scores Output: Complete statistical validation results

    Core Hypothesis Testing:

    1. H1: Discriminative Validity
    2. T-test: dignity texts vs tribalism texts
    3. Effect size calculation (Cohen's d)
    4. Confidence intervals and power analysis

    5. Target: p < 0.05 for separation

    6. H2: Ideological Agnosticism

    7. ANOVA: conservative vs progressive within dignity category
    8. ANOVA: conservative vs progressive within tribalism category
    9. Post-hoc tests for specific comparisons

    10. Target: p > 0.05 (no significant ideological difference)

    11. H3: Ground Truth Alignment

    12. Correlation analysis: predicted vs expected patterns
    13. Alignment scores for extreme controls (target >0.8)
    14. Balanced scoring validation for mixed controls (0.4-0.6)
    15. Pattern matching accuracy assessment

    Interrater Reliability Analysis (NEW):

    1. Core Reliability Metrics:
    2. Intraclass Correlation Coefficient (ICC): Absolute agreement between LLMs
      • Two-way mixed effects model, absolute agreement
      • 95% confidence intervals
      • Target: ICC > 0.75 (excellent agreement)
    3. Cronbach's Alpha: Internal consistency across LLM "raters"
      • Target: Œ± > 0.70 (acceptable reliability)
    4. Fleiss' Kappa: Multi-rater agreement for categorical judgments

    5. Pearson Correlations: All pairwise LLM correlations with confidence intervals

    6. LLM-Specific Reliability:

    7. Coefficient of Variation (CV): Per-text consistency across LLMs
    8. Standard Error of Measurement (SEM): Precision of LLM measurements
    9. Outlier Detection: Identify systematically deviating LLMs

    10. Systematic Bias Analysis: Check for consistent LLM differences

    11. Framework Robustness Testing:

    12. Cross-LLM Hypothesis Validation: Test if all LLMs support same conclusions
    13. Consensus Scoring: Weight LLM outputs by reliability
    14. Minimum Viable LLM Set: Determine optimal LLM combination
    15. Test-Retest Reliability: Multiple runs for stability assessment

    Deliverable: statistical_results.json with complete analysis results

    Phase C: Comprehensive Visualization Generation

    Input: Statistical results and analysis dataset Output: Publication-ready visualizations

    Core Visualizations:

    1. Discernus Framework Analysis:
    2. Individual circular coordinate plots for each text
    3. Comparative dignity vs tribalism positioning

    4. Interactive plots with hover details and text excerpts

    5. Hypothesis-Specific Charts:

    6. H1: Box plots with statistical annotations (p-values, effect sizes)
    7. H2: Grouped comparisons for ideological neutrality

    8. H3: Alignment heatmaps and correlation scatter plots

    9. Statistical Results:

    10. Forest plots for effect sizes with confidence intervals
    11. P-value visualization with significance thresholds
    12. Distribution plots for normality assessment

    Interrater Reliability Visualizations (NEW):

    1. Agreement Analysis:
    2. Bland-Altman Plots: LLM agreement with bias detection lines
    3. Correlation Matrix Heatmaps: All pairwise LLM correlations

    4. ICC Confidence Interval Plots: Reliability ranges by framework well

    5. Consistency Assessment:

    6. Box Plots by LLM: Individual LLM score distributions
    7. Coefficient of Variation Charts: Per-text reliability scores

    8. Outlier Detection Plots: Highlight unreliable measurements

    9. Multi-LLM Integration:

    10. Consensus Coordinate Plots: Combined LLM positions with uncertainty bands
    11. Individual LLM Overlays: Show all analyses simultaneously
    12. Reliability-Weighted Visualizations: Emphasize high-agreement regions

    Deliverable: HTML/PNG/SVG visualization files with interactive elements
